# Changelog

Формат: для регламента проекта (п. 7.2) фиксируются заметные изменения.

## [Unreleased]

- Подготовка пакета документов для согласования бота с администрацией (регламент).
- Опциональное логирование в файл (`LOG_TO_FILE`) для передачи администрации.
- Docker Compose с PostgreSQL для миграции с SQLite на проде.

## [1.0.0] — 2026-04-03

- Панель «Подать заявку» в `PANEL_CHANNEL_ID`, модалка заявки.
- Модерация в `MOD_CHANNEL_ID`: Approve (с модалкой способа приёма) / Reject.
- Доступ рекрутеров по ролям `APPROVER_ROLE_IDS`.
- Выдача всех ролей `TARGET_ROLE_IDS`, ник по `NICKNAME_PATTERN`.
- Аудит в `AUDIT_CHANNEL_ID`, пинг роли `ACADEMY_ROLE_ID`.
- Ежедневная статистика одобрений в `DAILY_STATS_CHANNEL_ID` (cron + таймзона).
- SQLite + Prisma, очистка slash-команд при старте (не критично при ошибке доступа).
