# Безопасность

- **Токен бота** хранится только в переменных окружения на хосте (файл `.env`, не в git).  
- В репозитории — только `.env.example` **без** секретов.  
- При утечке токена: немедленно **Reset Token** в [Discord Developer Portal](https://discord.com/developers/applications) → Bot.  
- Бот не обращается к внешним API, кроме Discord.  
- Исходный код не обфусцирован; поведение определяется репозиторием и конфигурацией `.env`.
